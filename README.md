##Black Hole Analysis

This project explores the relationship between black hole mass and distance using Python.

## What I did
- Created a dataset of stellar and supermassive black holes
- Visualized mass vs distance using matplotlib
- Used a logarithmic scale to better understand differences in size

## Key Insight
Supermassive black holes are significantly larger and typically located much farther away than stellar black holes.

## Tools Used
- Python
- matplotlib
- numpy

## Why I made this
I am interested in space science and wanted to start exploring real data using Python.
